# EXCEL_Sales_Dashboard
EXCEL dashboard practice. Using sumifs and Vlookup to make a quick dashboard, and then setting powerquery to automate monthly data processing. Final dashboard loading data from Powerquery and finishing with PivotTable.

## Profile Explanation:

6月_每日成交数据汇总：6月的成交数据（未处理）

历史数据：4月和5月的成交数据（已处理）

月度销售监控：Dashboard。可使用筛选器进行月份、地区的筛选

销售人员表-截止7月1日：用于调取销售的地区、省份、小组数据，使用销售人员编号与销售数据进行表连接
